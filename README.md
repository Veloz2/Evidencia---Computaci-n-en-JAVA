# Evidencia---Computaci-n-en-JAVA
## Sobre el proyecto:
    Este proyecto es un simulador de un gestor basico de  sistema de citas medicas. El sistema permite la alta de doctores, pacientes y la programacion de las citas medicas, con el fin de garantizar con los requisitos necesarios para completar la actividad.

## Funciones Principales
- Control de acceso: modulo de Login para el admin
- Gestion de entidades: Alta, registro con id unicos.
- Validacion de datos: evitar citas con pacientes inexistentes.


## Metodologia:
- Diagrama de flujo: Con el fin de visualizar la toma de decisiones y ciclos en este casi de uso raptor
- Pseucodigo: Implementado en PSeInt para validar la logica de programacion estructurada
- Diagrama de clases (UML): Sirve para dar estructura a los objetos necesarios para que funcione adecuadamente el proyecto, incluye Herencia y Polimorfismo.

## Software Necesario:
- PSeInt
- Raptor
- Java
